# Simple Mode UX

The default screen should offer only:

1. Tạo app từ ý tưởng của tôi
2. Tự tìm trend rồi tạo app
3. Mở dự án đang làm

Simple Mode hides worker, port, queue, Redis, Docker, migrations, deterministic mode, Codex, E2E, and API base URL. Those controls belong in Advanced Mode.

## Main Flow

Nhập ý tưởng -> Chọn loại app -> Chọn kiếm tiền -> Chọn công nghệ đơn giản -> Bắt đầu tạo app -> Xem tiến trình -> Xem APK/source/report.

## Writing Style

- Use Vietnamese product language, not developer terms.
- Explain outcomes, not implementation.
- Keep advanced concepts linked but visually secondary.
- Always state that human approval is required before store release.
