# Vietnamese Product UX Guide

ForgeTrend should feel like a friendly Vietnamese product, not a translated developer console.

## Defaults

- Default dashboard language: Tiếng Việt.
- Simple Mode first.
- Advanced Mode available but secondary.
- Explain outputs as APK, source, báo cáo, store assets.
- Avoid exposing queue, worker, port, Redis, migrations, and provider internals in the normal flow.

## Tone

Use clear Vietnamese:

- "Tạo app"
- "Xem tiến trình"
- "Cần con người review"
- "Mở artifact"
- "Cần cấu hình thêm"

Avoid raw technical labels unless in Advanced Mode.
