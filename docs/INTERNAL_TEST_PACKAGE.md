# Internal Test Package

ForgeTrend can create a local test folder for human internal testing:

```txt
internal_test_package/
  app-debug.apk
  README_FOR_TESTER.vi.md
  STORE_LISTING_DRAFT.vi.md
  PRIVACY_REVIEW_CHECKLIST.vi.md
  SCREENSHOT_PLAN.md
  RELEASE_BLOCKERS.md
```

This package is for manual review and internal testing only. ForgeTrend does not upload it to any store.
