# Hướng dẫn Simple Mode

Simple Mode chỉ giữ các việc người dùng phổ thông cần:

- Tổng quan
- Tạo app
- App ứng viên
- Kết quả / Artifact
- Cài đặt
- Trợ giúp

Các phần kỹ thuật như worker, provider, queue, doctor, logs và plugin nằm trong Advanced Mode.

## Tạo app nhanh

1. Mở `Tạo app`.
2. Nhập ý tưởng app.
3. Chọn ngôn ngữ app.
4. Chọn có kiếm tiền mô phỏng hay không.
5. Bấm `Bắt đầu tạo app`.

ForgeTrend sẽ tự research, tạo blueprint, sinh app, QA, quality gate và tạo report.
